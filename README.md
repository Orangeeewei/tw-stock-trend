# 台股趨勢日報

每個交易日收盤後自動產生的台股趨勢報告:**資金流向哪些產業、誰在帶頭漲、同產業還有誰沒漲**。
給沒有技術分析背景的人看——所有指標只用籌碼(三大法人)、月營收、價格位階、量能,並附白話解釋。

📰 **最新報告**:https://orangeeewei.github.io/tw-stock-trend/

## 報告內容

1. **大盤紅綠燈** — 加權指數是否站上 60 日線,決定補漲策略該不該執行
2. **產業熱度排行** — 各產業近 20 日表現(成分股中位數),看資金往哪去
3. **強勢產業領頭羊** — 創 60 日新高的風向標
4. **補漲候選** — 熱門產業 + 落後同業 + 低基期 + 甦醒跡象,附 0~100 進場分數:
   產業熱度 25 + 法人動向 30 + 營收動能 20 + 位階量能 25

## 資料來源(全部免費公開)

- 臺灣證券交易所:每日收盤行情(MI_INDEX)、三大法人買賣超(T86)、上市月營收
- 證券櫃檯買賣中心:上櫃行情、上櫃三大法人、上櫃月營收

涵蓋上市 + 上櫃 4 碼個股(排除 ETF),報告中上櫃標「櫃」。

## 自動化

GitHub Actions 於每個交易日台北時間 17:00 執行:更新資料 → 產生報告 →
發佈到 GitHub Pages → 推播摘要到 Discord(設定 `DISCORD_WEBHOOK_URL` secret 即啟用)。
歷史資料庫存於 Actions cache,遺失時自動重新回補。

## 本機使用

```bash
python3 main.py backfill --days 130   # 回補歷史資料
python3 main.py daily                 # 更新資料 + 產生報告
```

純 Python 標準庫,零外部依賴。報告輸出到 `reports/` 與 `docs/`。
